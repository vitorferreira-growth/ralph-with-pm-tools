## CloudWalk/Fintech Terms

| Term | Definition |
|------|------------|
| **Merchant** | Business that accepts payments via CloudWalk |
| **Acquirer** | CloudWalk's role - processes card transactions |
| **Issuer** | Bank that issued the customer's card |
| **Settlement** | Daily batch transfer of funds to merchant |
| **Chargeback** | Disputed transaction reversed to customer |
| **MDR** | Merchant Discount Rate - fee per transaction |
| **POS** | Point of Sale terminal |
| **mPOS** | Mobile POS (phone-based payment) |
| **PIX** | Brazilian instant payment system |
| **Boleto** | Brazilian payment slip |
